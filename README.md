# 🧮 Interactive AI Math Learning (Std 1–3)

An interactive math learning website designed for students of Standard 1 to Standard 3. The platform helps children learn basic mathematics in a simple, fun, and engaging way.

## 🚀 Features

### 📚 Learning Topics
- Counting Numbers
- Addition
- Subtraction
- Multiplication
- Division
- Number Comparison
- Missing Numbers
- Shapes
- Tables
- Simple Word Problems

### 🤖 AI Math Helper
- Solve Basic Math Questions
- Easy Step-by-Step Answers
- Child-Friendly Learning

### 🎯 Progress Tracking
- Learning Points
- Topics Covered
- Beginner Level

### 🎨 User Interface
- Simple Design
- Mobile Friendly
- Interactive Learning Experience

## 🛠️ Technologies Used
